Swap the Drop

Puts you in control of your favorite song.

Controls:

  • mouse click - start playing from the clicked beat
  • space - start and stop playing the current driver (space for all drivers)
  • arrows - control song cursor velocity for the current driver (shift/arrow for all drivers)
  • w/a/s/d - move the song cursor for the current driver (shift w/a/s/d for all drivers)
  • < > ? - controls the master beat period
  • b - toggle between beats and tatums
  • shift+number - bookmark a beat
  • command - cycle the though drivers
Drag to connect beats. See 'help' for more info
